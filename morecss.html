Random CSS Rules on Display!

width is a property that sets the width of an element. This does not include padding, boarders, or margins.
max-width is an element that defines the largest width an element is able to 'stretch' to. It has no effect if the width is below the value max-width is set to.
min-width is an element that defines the smallest width an element is permited to 'shrink' to. It has no effect if the width is above the value min-width is set to.

box-shadow attaches one or more shadows to an element.

The opacity property is used to adjust the transparency or opacity of an element.
